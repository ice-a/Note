# marktext

引用 两下回车退出

ctrl+p  当前文件搜索

ctrl +j 呼出侧边栏

ctrl + alt +b 呼出并行文件栏

crtl +Tab 循环并行文件

CTRL +alt +S 切换为代码模式

crtl alt T 打字机模式（有啥用码。。

crtl shift j 专注模式

ctrl o 打开文件

ctrl s 保存

ctrl shift o 打开目录

ctrl +l 超链接
