# MarkText

引用如何退出？

按两下回车退出

ctrl+P  搜索文件

ctrl+F   当前文件搜索

ctrl+shift+F 在文件夹中搜索

ctrl+shift+P 命令行

ctrl+J 呼出侧边栏

ctrl+shift+b 呼出并行文件栏

crtl+Tab 循环并行文件

ctrl+E    源代码模式

crtl+shift+g 打字机模式

crtl+shift+J 专注模式

ctrl+O 打开文件

ctrl+S保存

ctrl+shift+O 打开目录

ctrl+L 超链接
